# Frontend Design Skill

Creates distinctive, production-grade frontend interfaces with high design quality.

## Overview

This skill guides creation of exceptional frontend interfaces that avoid generic "AI slop" aesthetics. It generates creative, polished code and UI design with exceptional attention to aesthetic details and creative choices.

## When to Use

Use this skill when you need to build:
- Web components and pages
- Landing pages and websites
- Dashboards and applications
- React/Vue/HTML components
- Styled/beautified web UIs
- Any web interface requiring high design quality

## Design Principles

### Design Thinking
- **Purpose**: Understand the problem and audience
- **Tone**: Choose bold aesthetic direction (minimal, maximalist, retro, organic, luxury, brutalist, etc.)
- **Constraints**: Consider technical requirements and accessibility
- **Differentiation**: Create something unforgettable

### Aesthetic Guidelines

**Typography**
- Choose distinctive, characterful fonts
- Avoid generic choices (Arial, Inter, Roboto)
- Pair display fonts with refined body fonts

**Color & Theme**
- Commit to cohesive aesthetic
- Use dominant colors with sharp accents
- Implement CSS variables for consistency

**Motion**
- High-impact animations for page loads
- Staggered reveals and scroll-triggering
- Surprising hover states and transitions

**Spatial Composition**
- Unexpected layouts and asymmetry
- Grid-breaking elements
- Generous negative space or controlled density

**Backgrounds & Visual Details**
- Create atmosphere with gradients, textures, patterns
- Dramatic shadows and layered transparencies
- Context-specific decorative elements

## Output Quality

Generates production-grade code that is:
- Functional and well-structured
- Visually striking and memorable
- Cohesive with clear aesthetic vision
- Meticulously refined in every detail

```bash

# Personal (available in all projects)
mkdir -p ~/.claude/skills
cp -r SKILLS/skills/frontend-design ~/.claude/skills/
```
